# CRUD Flow Skill

Standardize Create, Read, Update, Delete flows.

## When to use

- User asks for CRUD UI/page
- Input includes create, edit, delete, detail, list

## Steps

1. Create list page (use list-flow)
2. Create detail page
3. Create form (use form-flow)
4. Connect API for CRUD
5. Handle optimistic update, rollback on error
6. Add confirm dialog for delete

## Output

- List page (React)
- Detail page (React)
- Form (React)
- API logic

## Constraints

- Use React + TypeScript
- Use Tailwind CSS
- Follow design system

## Anti-pattern

- Do not duplicate API calls
- Avoid inline styles
