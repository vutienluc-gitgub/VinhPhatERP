# ERP Flow - Textile Business

## Order Flow

1. Select customer
2. Select fabric lot
3. Input quantity (meter)
4. Check stock
5. Create order
6. Deduct inventory

## Inventory

- Track by lot
- Fields:
  - code
  - color
  - width
  - gsm
  - quantity

## Rules

- Cannot create order if not enough stock
- Always update inventory after order
